### OLD LAYOUT ( MODULAR STYLE )

```
Nixpkgs
  |
  |------- Machine
  |
  |------- Role
  |         |
  |         |---- Conf-Utils ( system utilities )
  |         |---- Conf-Wlr   ( wayland utilities )
  |         |---- Conf-x11   ( xorg utilities )
  |         |---- Wall       ( wallpaper ) 
  |         |---- Xpm        ( icons )
  |
  |
  |------- User
            |
            |-------- Config
            |-------- Nur ( Nix User Repository )
            |-------- Scripts
  
```
